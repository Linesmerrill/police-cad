Release Log

Release 0.8.7

Mobile
PC

Release 0.8.7 adds firearm status, adds Patreon link and fixes warrant display bug

- You can now add or modify a firearm license on each civilian.
- The new statuses are: None, Valid, and Revoked.
- This status will also be displayed for officers when they look up civilians in the police database.
- The status defaults to None, so be sure to update this for each civilian accordingly.

- One of our users pointed out a bug where sometimes it would not show the warning for a warrant.
- This will now properly show warrant status ran by NCIC when searching a civilian in the database.

For more info on this patch view our release log: Patch v0.8.7
